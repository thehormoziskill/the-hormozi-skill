# Public Skill Repository

Installable skill package for Codex.

## Install

### macOS / Linux

```bash
curl -fsSL https://raw.githubusercontent.com/thehormoziskill/the-hormozi-skill/main/installers/bootstrap.sh | bash
```

### Windows

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -Command "iwr https://raw.githubusercontent.com/thehormoziskill/the-hormozi-skill/main/installers/bootstrap.ps1 -UseBasicParsing | iex"
```

The install places the skill at:
- `~/.codex/skills/the-hormozi-skill`

Restart Codex after install.
